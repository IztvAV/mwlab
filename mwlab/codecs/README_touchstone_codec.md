### 1. –ê–≤—Ç–æ–º–∞—Ç–∏—á–µ—Å–∫–∞—è –≥–µ–Ω–µ—Ä–∞—Ü–∏—è –∫–æ–¥–µ–∫–∞ –∏–∑ –¥–∞—Ç–∞—Å–µ—Ç–∞

```python

# TouchstoneCodec(x_keys=2, y_channels=8[imag,real], freq_pts=256, ports=2)
```

### 2. –ö–æ–¥–∏—Ä–æ–≤–∞–Ω–∏–µ –ø—Ä–∏–º–µ—Ä–∞ –≤ —Ç–µ–Ω–∑–æ—Ä—ã
